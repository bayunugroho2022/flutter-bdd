# how to run 

## get dpendency
``` flutter pub get ```

## install node
``` npm install cucumber-html-reporter ```

## test login feature
``` flutter drive --driver=test_driver/config/login/app_login.dart ```

## see report
``` open node_modules/cucumber-html-reporter/samples/html_reports/cucumber_report_bootstrap.html ```
